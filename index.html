📌 CARA TAMBAH GAMBAR KE GALLERY

Buka repository kamu di GitHub, klik Add file → Create new file, ketik images/ lalu nama file (contoh: images/grafik1.png) — ini otomatis buat folder images.
Atau klik Add file → Upload files, lalu masuk ke folder images dan upload gambar di sana.
Buka file index.html di GitHub, klik ikon pensil (edit), cari bagian gallery dan copy contoh blok gambar di komentar.
Ganti images/grafik1.png dengan nama file yang kamu upload, isi judul dan tag, lalu Commit changes. Selesai! 🎉
